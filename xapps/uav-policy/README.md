# uav-policy xApp

This package contains the **policy engine** for UAV-related O-RAN use cases.

Key file:

- `src/uav_policy/policy_engine.py` — data models and algorithms.
- `tests/test_policy_engine.py` — unit tests that document expected behavior.
- `src/uav_policy/main.py` — simple CLI entrypoint for local experiments.
