# Command: /design-uav-policy-step

## Purpose

Guide a single, small design+implementation step for the UAV policy.

## Steps

1. Restate the specific behavior change or feature.
2. Identify the affected data structures and functions in
   `policy_engine.py`.
3. Update or add tests in `tests/test_policy_engine.py`.
4. Modify code in `policy_engine.py` to satisfy the tests.
5. Suggest a follow-up step if needed.
